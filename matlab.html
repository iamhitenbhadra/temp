SOLUTION OF SECOND ORDER ORDINARY DIFFERENTIAL EQUATIONS WITH CONDITIONS

Objective: To find solutions of linear differential equations of second and higher-order with constant coefficients and boundary conditions.
Syntax for general differential equation solutions:
second_order_ode.m
clc;
clear;
syms y(x)
DE = diff(y,x,3) == f(x) - a1*diff(y,x,2) - a2*diff(y,x) - a3*y;
Sol_y(x) = dsolve(DE)
Syntax with Initial Conditions:
ode_with_conditions.m
clc;
clear;
syms y(x)
DE = diff(y,x,3) == f(x) - a1*diff(y,x,2) - a2*diff(y,x) - a3*y;
dy = diff(y,x);
d2y = diff(y,x,2);
condition1 = y(x0) == a;
condition2 = dy(x0) == b;
condition3 = d2y(x0) == c;
condition = [condition1 condition2 condition3];
Sol_y(x) = dsolve(DE, condition)

MODIFIED EULER'S METHOD

Objective: Numerical approach to approximate solutions of first-order ordinary differential equations.
modified_euler.m
clc;
clear all;
f = @(x, y) x + y;
x0 = 0;
y0 = 1;
h = 0.2;
x1 = x0 + h;
% Euler Formula
y1 = y0 + h * f(x0, y0);
fprintf('Eulers Formula %f\n', y1);
fprintf('Iteration y-value\n');
for k = 1:5
    y1 = y0 + (h/2) * (f(x0, y0) + f(x1, y1));
    fprintf('%d       %f\n', k, y1);
end
fprintf('\nFinal value at x = %.1f is y = %.6f\n', x1, y1);

RUNGE-KUTTA METHOD (4th ORDER)

Objective: Solve initial value differential problems with high computational precision stability.
runge_kutta_4.m
clc;
f = @(x,y) 3*x + y/2; % Modify target inline function here
x = 0;
y = 1;
h = 0.2;
X = 0.2;
while X - x > 0
    fprintf('value of y at x = %0.2f is %f \n', x, y)
    k1 = h * f(x, y);
    k2 = h * f(x + h/2, y + k1/2);
    k3 = h * f(x + h/2, y + k2/2);
    k4 = h * f(x + h, y + k3);
    k = (1/6) * (k1 + 2*k2 + 2*k3 + k4);
    x = x + h;
    y = y + k;
end

Part B: Angle Matrix Interface between Intersecting Environments
angle_between_surfaces.m
syms x y z
phi = x^2 + y^2 + z^2;
psi = x+y+z;
X = [x y z];
p = [1 -1 2];
grad1 = gradient(phi, X);
grad2 = gradient(psi, X);
grad1p = subs(grad1, X, p);
grad2p = subs(grad2, X, p);
n1 = norm(grad1p); % Finds |grad phi|
n2 = norm(grad2p); % Finds |grad psi|
A = eval(grad1p/n1);
B = eval(grad2p/n2);
theta = acos(dot(A, B))

DIVERGENCE, CURL, & FIELD BEHAVIOR

Objective: Analyze directional fluid vectors to classify whether a field is Solenoidal (div = 0) or Irrotational (curl = 0).
Part A: Divergence (Solenoidal Test)
vector_divergence.m
syms x y z
V = [x^2, y^2, z^2]; % Input Vector field
X = [x y z];
p = [1 1 1];
div = divergence(V, X);
divp = subs(div, X, p)
if(divp == 0)
    display('since divp is zero, hence Vector field is Solenoidal')
else
    display('since divp is non zero, hence Vector field is not Solenoidal')
end
Part B: Curl Formulation (Irrotational Test)
vector_curl.m
syms x y z
V = [x^2, y^2, z^2];
X = [x y z];
p = [1 1 1];
curl_val = curl(V, X);
curlp = subs(curl_val, X, p)
if(curlp == [0 0 0])
    display('since curlp is zero, hence Vector field is irrotational')
else
    display('since curlp is non zero, hence Vector field is not irrotational')
end

TAYLOR'S SERIES APPROXIMATION METHOD

Objective: Track recursive expansion orders for solving higher degree nonlinear operational forms.
taylor_series_method.m
clc;
clear all;
syms y x ;
dy = x - y^2; % Define function f(x,y)
ix = 0;       % initial x0
iy = 1;       % initial y0
a1 = subs(dy, [x, y], [ix, iy]);
d2y = diff(dy, x) + diff(dy, y) * dy;
a2 = subs(d2y, [x, y], [ix, iy]);
d3y = diff(d2y, x) + diff(d2y, y) * dy;
a3 = subs(d3y, [x, y], [ix, iy]);
d4y = diff(d3y, x) + diff(d3y, y) * dy;
a4 = subs(d4y, [x, y], [ix, iy]);
x1 = 0.1;     % Evaluation point target
ans = iy + (x - ix)*a1 + ((x - ix)^2*a2)/factorial(2) + ((x - ix)^3*a3)/factorial(3)+((x - ix)^4*a4)/factorial(4);
fprintf('hence y(%.2f)=%.6f\n', x1, ans);
